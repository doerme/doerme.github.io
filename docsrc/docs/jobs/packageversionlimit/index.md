# package.json 版本号规范

| 格式 | 作用 |
|:------|:------|
| 1.2.3 | 指定匹配1.2.3版本 |
| ^1.2.3 | 左第一位非0位不可改变，第二位及之后可以为更高版本 |
| last | 当前发布版本，常见的还有 next stable beta canary |
| ^1.x | 匹配 >=1.0.0 且 <2.0.0 |
| 1 | 匹配 >=1.0.0 且 <2.0.0 |
| 1.2 | 匹配 >=1.2.0 且 <1.3.0 |
| ~1.2.3 | 匹配 >=1.2.0 且 <1.3.0 |
| > = < | 同字面表达意思 |
